
# Superstore Sales Insights

## Overview
This project explores retail sales data to understand how sales, profit,
products, and regions contribute to overall business performance.
The goal is to extract meaningful insights that support data-driven decisions.

## Dataset
Superstore sales dataset in CSV format.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Questions Explored
- How do sales and profit change over time?
- Which categories and products generate the highest revenue?
- How does sales performance vary across regions?

## Outcome
The analysis provides clear insights into sales trends and business performance,
which can help stakeholders make informed decisions.
